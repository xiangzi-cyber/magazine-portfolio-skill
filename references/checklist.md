# 质量检查清单(Checklist)

生成或修改作品页 HTML 后逐项对照。规则正文见 `rule-gates.md`;本文件只保留检查项。

---

## P0 · 必过

### P0-0 · 类名预检 [G5/P0-5]

- [ ] 页面从 `assets/template.html` 或已验证母版页起步。
- [ ] 主结构使用 `.hero` / `.act-cover` / `.section` / `.outro`。
- [ ] sections.md 用到的类在模板 `<style>` 块里有定义。
- [ ] 没有发明新类名替代模板类名系统。
- [ ] 字体继承母版 token。

### P0-1 · 4 章节结构 [G5]

- [ ] 章节顺序为项目背景 → 设计判断 → 落地交付 → 方法沉淀。
- [ ] 每章节有独立 `.act-cover`(章节四例外,直接做 `.outro`)。
- [ ] `id="act-1"` / `id="act-2"` / `id="act-3"` / `id="act-4"` 都设置。
- [ ] 每段 `.section` 归属正确。

### P0-2 · 段落级信息密度 [G5]

- [ ] `.chrome` 段顶元数据。
- [ ] `.kicker` 短前缀。
- [ ] `<h3>` 主标。
- [ ] `.h-sub` 副标。
- [ ] `<p class="body">` 至少 2 段副文。
- [ ] `.anchor-quote` 或 `.callout` 至少一种。

### P0-3 · 必改占位符 [G5]

- [ ] `<title>` 替换。
- [ ] `<meta name="description">` 替换。
- [ ] `.topbar .brand` 替换。
- [ ] `.hero h1` / `.hero .meta` / `.hero .subtitle` / `.hero .lead` 替换。
- [ ] grep `[` 确认所有方括号占位符清零。

### P0-4 · 主题色 [G5]

- [ ] 使用 5 套预设之一。
- [ ] `:root{}` 6 行变量整体替换。
- [ ] 没有混搭主题。

### P0-5 · 终稿口吻隔离 [G2/G5/G6/P0-1]

- [ ] HTML 内容架构稿已完成口吻转写。
- [ ] 最终 HTML 无内部审稿提示词。
- [ ] 最终 HTML 无内部审稿组件。
- [ ] 若删掉内部字段后页面价值不清楚,回到 M7。

### P0-6 · 术语一致 [G2/G5]

- [ ] 同一术语全文只用一个版本。
- [ ] 中英文术语首次出现后不混译。

### P0-7 · 保留用户口气词 [G2]

- [ ] 用户原词未被 corporate 术语替换。
- [ ] 时间、数字、协作边界未被改写。

### P0-8 · 锚点金句格式 [G5]

- [ ] 每段(除 `.outro`)有 1 句 `.anchor-quote`。
- [ ] 中英对仗。
- [ ] 小于 30 字。
- [ ] 不直接重复主标。

### P0-9 · 前后一致性 [G2/G5]

- [ ] 交付物数量前后一致。
- [ ] "我做的"vs"团队做的"明确区分。
- [ ] 导航栏、章节封面、段顶元数据命名一致。
- [ ] 时间线无未解释空白。

### P0-10 · 视觉 QA [G6/G7]

- [ ] 同类组件统一修。
- [ ] 版心一致。
- [ ] 标题不被压窄。
- [ ] 右侧无大片空白。
- [ ] 线框不过量。
- [ ] 收尾页光感温和。
- [ ] 桌面/手机都过。

### P0-11 · 图片冻结任务 [G4/P0-2]

- [ ] 只替换图片路径、容器、alt、极短图片标签。
- [ ] 未重构页面。
- [ ] 已核查落位清单。
- [ ] 复合图需求已另开任务。
- [ ] 交付后只汇报图片动作。

### P0-11A · 图片入库 [G3]

- [ ] 原图不动。
- [ ] 先筛选再压缩。
- [ ] 命名可追溯。
- [ ] 生成落位表。
- [ ] 缺口分级。
- [ ] 压缩后验证清晰度、尺寸、文件大小和浏览器加载。

### P0-12 · 生成后视觉验收 [G6/P0-3/P0-4]

- [ ] 图片语义对应。
- [ ] 图片路径存在且加载正常。
- [ ] 重复图片脚本审计通过。
- [ ] 同组成果不连铺。
- [ ] 大白框/空白截图已处理。
- [ ] 证据卡图片区、标题、说明间距正常。
- [ ] 同组图片高度一致。
- [ ] 内容图未被错误裁切。
- [ ] 窄屏不横溢。
- [ ] 浏览器实测 broken image = 0、overflowX = 0。

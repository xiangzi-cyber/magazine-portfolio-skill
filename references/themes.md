# 主题色预设(Themes)

5 套精心调配的主题色板,从 [guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill) 直接继承,保证"电子杂志 × 电子墨水"美学不垮。

**不允许用户自定义颜色** —— 色彩搭配错了画面瞬间变丑,只从以下预设中挑选。

> **整站统一原则**:作品集所有页面用同一套主题色 ——
> 首页 / 关于页 / 项目页 1 / 项目页 2 / 项目页 N **不要换主题**。
> 整站颜色一致 = 视觉品牌记忆;颜色乱跳 = 招聘方认知摩擦。

---

## 使用方法

1. 基于求职方向推荐一套,或问用户选哪一套
2. 打开 `assets/template.html` 的 `<style>` 块
3. 找到开头的 `:root{` 块
4. **整体替换**标有"主题色"注释的 6 行(`--ink` / `--ink-rgb` / `--paper` / `--paper-rgb` / `--paper-tint` / `--ink-tint`)
5. 其他 CSS 都走 `var(--...)`,无需任何其他改动

## 🖋 墨水经典 (Monocle 默认)

**适合**:通用作品集 / 商业岗位 / 互联网大厂 / 不知道选啥的默认。
**调性**:纯墨黑 + 暖米白,杂志感最强,Monocle / Apricot / A Book Apart 风。

```css
--ink:#0a0a0b;
--ink-rgb:10,10,11;
--paper:#f1efea;
--paper-rgb:241,239,234;
--paper-tint:#e8e5de;
--ink-tint:#18181a;
```

## 🌊 靛蓝瓷 (Indigo Porcelain)

**适合**:科技 / 研究 / AI 产品 / 数据 / 工程师 / 技术品牌设计师作品集。
**调性**:深靛蓝 + 瓷白,冷静、理性、有深度,像学术期刊或蓝印花瓷器。

```css
--ink:#0a1f3d;
--ink-rgb:10,31,61;
--paper:#f1f3f5;
--paper-rgb:241,243,245;
--paper-tint:#e4e8ec;
--ink-tint:#152a4a;
```

## 🌿 森林墨 (Forest Ink) ★ 本 skill 默认推荐

**适合**:公益 / 教育 / 文化 / 可持续 / 自然 / 内容创作 / 非虚构。
**调性**:深森林绿 + 象牙,沉稳、有呼吸感,像旧版《国家地理》。

```css
--ink:#1a2e1f;
--ink-rgb:26,46,31;
--paper:#f5f1e8;
--paper-rgb:245,241,232;
--paper-tint:#ece7da;
--ink-tint:#253d2c;
```

> 这是箱子第一份长滚动作品页(火种车 LOGO)用的主题。
> 验证过的稳定组合,适合 "AI 教育 / 公益 / 教育下乡 / 文化项目 / 可持续设计" 类作品集。

## 切换原则

- **整站只用一套主题** —— 作品集首页 / 关于页 / 所有项目页保持一致
- WebGL hero 背景的颜色由 `--ink` / `--paper` 自动适配,**不需要单独调整 shader**
- `currentColor` 驱动的 border / icon 会跟随 section 的 text color 自动适配
- `<title>` 文字和 `chrome` 元数据可以强化主题语义(例如牛皮纸配 "Vol.03 · 秋")

---

## ❌ 不要做的事

- ❌ **不允许混搭**(例如 ink 取墨水经典、paper 取沙丘)—— 彻底违和
- ❌ **不允许用户自定义 hex 值** —— 委婉拒绝并展示 5 套让选
- ❌ **不要逐处改 template.html 散落的颜色** —— 所有颜色走 var,改 :root 一处即可
- ❌ **不要在不同项目页用不同主题** —— 整站统一,加深品牌记忆
